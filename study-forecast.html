Summary

Reduced forecasting error (MAPE) by 18% using SARIMAX with holiday/seasonality features.

Problem

Predict weekly sales at category‑store level; reduce stockouts and overstock.

Data

  • Sources: POS transactions, calendar (holidays), promotions
  • Timeframe: Jan 2023 – Dec 2023
  • Size & key fields: ~2M rows; keys: date, store_id, category_id; y: sales

Approach

Baseline SARIMA → add exogenous regressors (promo, holiday dummies); rolling origin validation.

from statsmodels.tsa.statespace.sarimax import SARIMAX
model = SARIMAX(y, order=(1,1,1), seasonal_order=(1,1,1,52), exog=X).fit()

Results

MAPE ↓ 18%; identified 3 promo windows with largest elasticity.
